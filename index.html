사용 방법

기본 사용법
  1. 각 입력란에 캐릭터 정보를 입력하세요.
  2. 항목 추가로 원하는 만큼 줄을 추가할 수 있어요.
  3. 주석은 {{주석 내용}}을 넣으면 자동으로 완성됩니다.
  4. 우측 미리보기는 입력 시 자동 갱신됩니다. 수동 갱신은 코드 갱신을 누르세요.
  5. 티스토리 글쓰기 → 기본모드 → HTML 모드에서 코드를 붙여 넣으면 끝!
  6. 작업 내용은 저장 버튼으로 JSON 파일로 보관하세요.
TIP
  1. 본문 에디터의 블록은 드래그하여 순서를 바꿀 수 있습니다.
  2. 아이콘이 표시되지 않으면 티스토리 스킨 HTML의 </body> 바로 위에 Ionicons 스크립트를 추가해 주세요.
  3. HTML 모드로 글을 저장한 후 수정할 때는 사이트를 이용해주세요. 티스토리 에디터 내에서 기본 모드로 전환한 후 저장 시 코드의 일부분이 사라집니다
